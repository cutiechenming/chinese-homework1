解题思路

这道题考查根据拼音写汉字的能力，需要结合语境判断正确的字形：

1. 祭sì：祭祀祖先，所以是"祀"（祭祀）

3. 气势恢hóng：形容气势宏大，所以是"宏"（恢宏）

正确答案

解题思路

本题考查词语搭配和标点符号的使用：

【甲】处：

"保存"指使事物存在下去，不被破坏或消失（如"保存遗迹"）；

"保养"指保护调养（多用于物品、身体等）。此处搭配"祠堂文化遗迹"，应选"保存"。

【乙】处：

"吏瘦民肥"是直接引用的匾额文字，并非书籍或篇章标题，应用引号（""），而非书名号（《》）。

排除A、D（"保养"搭配不当），排除C（"《》"使用错误），所以正确答案是B。

作者：谭嗣同（清末维新派志士）

背景：戊戌变法失败后，谭嗣同等六人被捕，这首诗是谭嗣同在狱中所作，表达了他为变法献身的决心。

赏析：前两句借用张俭、杜根的典故，期望康有为、梁启超等维新志士能像张俭、杜根一样逃脱迫害，保存实力。后两句直抒胸臆，表现了自己为变法慷慨赴死的豪迈气概和坚定信念。"我自横刀向天笑"成为千古名句，展现了革命者视死如归的英雄气概。

"小月亮"：蕴含对故乡无穷的思念、无限的爱，以及追忆、惆怅、留恋、惋惜等复杂情感。

"国土"：蕴含对受难祖国的眷恋与忧虑（或"土地的忧郁"），以及对"永恒中国"的希望与坚定信念。

19.【学以致用】请说明链接材料中艾青笔下的"火把"有何特点，并结合材料一、二分析这些特点形成的原因。

特点：美丽、耀眼、热情、象征团结与光明（或能照亮黑暗、驱赶阴影；凝聚人群，激发团结与前进的力量）。

原因：

材料一：根据"美不自美，因人而彰""没有心灵的映射，是无所谓美的"，火把的"美"源于艾青的审美心灵创造。他将火把与抗战背景下人民对光明、团结的渴望结合，赋予其情感与意蕴。

材料二：抗战时期民族危亡，诗人需回应时代命题。艾青的火把"搜索黑暗""卫护前进"，熔铸了对民族命运的关切与家国情怀，激发读者的忧患意识与民族认同。

（四）现代文阅读《暖心时刻》
20分

阅读白雪松的《暖心时刻》一文，完成20-24题。

文章讲述了出租车司机小陈帮助一位身患重病的空巢老人，满足她最后愿望的暖心故事。

20.请梳理文中老人情感变化的过程，补全下面的横线。（用文中词语填空，4分）

惭愧——①悲怆——②动情——高兴——③伤感——④惊愕——感动

21.联系上下文，根据要求赏析下面两个句子。

(1) 小陈把老人扶下来，把轮椅折叠好，放到自己肩上，又腾出另一只手搀扶老人，想背她下楼。

赏析：运用动作描写，通过"扶、放、腾、搀"等一系列动词生动形象地表现出小陈帮助老人下楼的情形，表现出小陈爱护老人、细心周到的人物形象。

(2) 只要得到一点触动，有些事就会从心底最深处升腾起来，像根绳索捆绑着自己，深深勒进脑海。

赏析："升腾"一词表现出老人内心的往事不受控制地涌上心头；"捆绑"表现出往事缠绕心头，不得解脱之感；"勒"突出了往事的印象之深。

22.文章第⑪节画线处环境描写有何作用？请结合文章内容简要分析。

1. 暗示时间推移，"曙光透出地平线"表明天色已明。

2. 渲染了明亮温暖的氛围，用"红云""云彩"烘托老人回忆过往后内心的释然与希望。

3. 推动小说情节发展，为后文她决定进医院面对手术做铺垫。

4. 暗示老人勇敢直面病魔，对未来充满希望，曙光象征新生与希望。

23.结合语境，揣摩第⑭节中两处"霞光"有何不同意蕴。

前一个"霞光"指的是自然的日光云霞。

后一个"霞光"意蕴深刻，指老人心中已燃起了生的希望，有了战胜病魔的勇气和信心。

24.结合文章内容，探究作品的主题。

小说通过出租车司机小陈帮助空巢老人的故事，表达了人与人之间互相理解和关爱的温情，呼吁人们关爱空巢老人，展现了善良、互助、理解、尊重等美好品质。

三、作文（55分+书写分5分）
60分

"和解"并非软弱，而是洞明世事后的强大抉择。苏轼以"也无风雨也无晴"与坎坷际遇和解，彰显超然气度；史铁生以赴北海看花与自身苦难和解，踏上救赎之路。真正的和解，是灵魂的成长，引领我们走向更辽阔的人生。

请结合自己的经历，以《与自己和解》为题，写一篇记叙文。

写作指导

1. 审题关键："与自己和解"意味着接受自己的不完美，放下内心的纠结和矛盾。

2. 选材建议：可以写学习压力、与父母矛盾、朋友关系、自我认知等方面的经历。

3. 结构安排：开头点题 → 叙述具体事件 → 描写内心挣扎 → 和解的过程 → 感悟升华。

4. 写作技巧：注意细节描写，尤其是心理变化过程；结尾要点明和解的意义。

作文写作要点

《与自己和解》写作要点：

立意角度

1. 接受自己的不完美（如学习成绩、外貌、性格等）

2. 放下过去的遗憾或错误

3. 与内心的焦虑、压力和解

4. 理解并接纳自己的独特之处

素材选择

1. 学习上：从"唯分数论"到关注全面发展

2. 成长中：接受自己的普通，找到自己的闪光点

3. 人际关系：理解父母、老师的期望，找到平衡点

4. 兴趣特长：接受自己的爱好不被他人理解

结构建议

1. 开头：引用材料或直接点题，引出"和解"主题

2. 主体：详细叙述具体事件，重点描写内心矛盾与挣扎

3. 转折：某个契机让你开始思考"和解"

4. 高潮：和解的过程和感悟

5. 结尾：升华主题，谈和解的意义和成长

注意事项

1. 必须是记叙文，要有具体事件和细节描写

2. 心理描写要细腻真实，展现变化过程

3. 字数不少于600字，注意卷面整洁

4. 文中不得出现真实的校名、地名和人名
